# QAP 4 – Java Data Persistence

A Java program that saves and reads Drug data to a text file and Patient data to a PostgreSQL database through a console menu.

## How it went

Started with the entity classes and file I/O then moved to the database side. Most of the time was spent on setup — getting PostgreSQL running, Maven configured, and a Java version mismatch sorted out. Once connected everything worked as expected.

## Questions

- Resources used: course lectures, AI assistance for debugging
- Classmate help: none
- Instructor help: none
- Difficulty: 2/5 — confident doing this again
